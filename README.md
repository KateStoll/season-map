# 🍂 Season Map

**Season Map** is a cozy seasonal activity app that helps people discover, plan, and share fall (and year-round) adventures.  

It combines seasonal bingo cards, an interactive calendar of events, and an illustrated “season map” to reduce FOMO and bring more joy to every season.  

---

## ✨ Vision
- Help people find local and at-home seasonal activities.  
- Make it fun and interactive with bingo/checklists, maps, and cozy animations.  
- Encourage sharing and community through playlists, photos, and challenges.  
- Grow beyond fall into all four seasons.  

---

## 🎯 MVP Features
- Seasonal bingo cards (Kids, Couples, Friends, Pets, Solo)  
- Interactive calendar (local + at-home activities)  
- Illustrated map with seasonal icons  
- Optional playlist linking (Spotify/YouTube)
